Interactive Haiku

Some haiku are now interactive using the button and buttons words to parameterize a haiku based on button / keyboard input. The mx and my words can be used to base a haiku on mouse location.

These haiku can only be seen in animated form using Google Chrome, Firefox, and Safari. Click on haiku marked with an 'int' to see the interactive animation.

The haiku is converted on the fly to a GLSL fragment shader.
